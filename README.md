## Задача 1

**Условие:**  
Создать класс для прямоугольника с методами вычисления площади и периметра.

**Решение:**  
Реализован класс Rectangle в файле geometry/rectangle.py с методами:
- area() – возвращает площадь (a * b)
- perimeter() – возвращает периметр (2 * (a + b))

**Результат:**

## Задача 2

**Условие:**  
Создать класс для треугольника с методами вычисления площади и периметра.

**Решение:**  
Реализован класс Triangle в файле geometry/triangle.py с методами:
- area() – возвращает площадь по формуле Герона
- perimeter() – возвращает периметр (a + b + c)
- info() – возвращает строку с информацией о фигуре

**Результат:**
tri = Triangle(3, 4, 5)
tri.area() = 6.0
tri.perimeter() = 12

## Задача 3

**Условие:**  
Создать класс для трапеции с методами вычисления площади и периметра.

**Решение:**  
Реализован класс Trapezoid в файле geometry/trapezoid.py с методами:
- area() – возвращает площадь ((a + b) / 2 * h)
- perimeter() – возвращает периметр (a + b + 2 * h)
- info() – возвращает строку с информацией о фигуре

**Результат:**
trap = Trapezoid(8, 4, 3)
