### Задача 1: Класс `Rectangle` (Прямоугольник)

**Условие:**  
Создать класс для прямоугольника с методами вычисления площади и периметра.

**Решение:**  
Реализован класс `Rectangle` с конструктором `__init__(self, a, b)` и методами:
- `area()` — возвращает площадь (`a * b`)
- `perimeter()` — возвращает периметр (`2 * (a + b)`)
- `info()` — возвращает строку с информацией о фигуре

**Результат:**
```python
rect = Rectangle(5, 8)
print(rect.info())      # Прямоугольник: a=5, b=8
print(rect.area())      # 40
print(rect.perimeter()) # 26
Задача 2: Класс Triangle (Треугольник)
Условие:
Создать класс для треугольника с методами вычисления площади и периметра.

Решение:
Реализован класс Triangle с конструктором __init__(self, a, b, c) и методами:

area() — возвращает площадь по формуле Герона

perimeter() — возвращает периметр (a + b + c)

info() — возвращает строку с информацией о фигуре

Добавлена проверка существования треугольника (неравенство треугольника).

Результат:

python
tri = Triangle(3, 4, 5)
print(tri.info())       # Треугольник: a=3, b=4, c=5
print(tri.area())       # 6.0
print(tri.perimeter())  # 12
Задача 3: Класс Trapezoid (Трапеция)
Условие:
Создать класс для трапеции с методами вычисления площади и периметра.

Решение:
Реализован класс Trapezoid с конструктором __init__(self, a, b, h) и методами:

area() — возвращает площадь ((a + b) / 2 * h)

perimeter() — возвращает периметр (a + b + 2 * h)

info() — возвращает строку с информацией о фигуре

Результат:

python
trap = Trapezoid(8, 4, 3)
print(trap.info())      # Трапеция: a=8, b=4, h=3
print(trap.area())      # 18.0
print(trap.perimeter()) # 18
